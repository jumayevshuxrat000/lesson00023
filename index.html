JavaScript: Map va Set Tushuntiruvchi Sayt

Map

Map bu JavaScriptdagi maxsus ma'lumot tuzilmasi bo'lib, u kalit-qiymat juftlarini saqlaydi. Kalitlar har qanday turdagi ma'lumot (shu jumladan object) bo'lishi mumkin.

Map xususiyatlari:

  • Kalitlar unikaldir, qaytarilmaydi.
  • Qiymatlar har qanday turdagi bo'lishi mumkin (raqam, string, object, va h.k.).
  • Metodlar: set, get, delete, has, va boshqalar.

Misol:

                   const map = new Map();
                   map.set('name', 'John');
                   map.set('age', 30);
                   console.log(map.get('name')); // John
                   console.log(map.has('age')); // true

Set

Set bu JavaScriptdagi boshqa bir ma'lumot tuzilmasi bo'lib, u faqatgina unikalligi kafolatlangan qiymatlarni saqlaydi. Shu sababli, qiymatlar qaytarilmaydi.

Set xususiyatlari:

  • Qiymatlar yagona (unikal) bo'lishi kerak.
  • Metodlar: add, delete, has, va boshqalar.

Misol:

                 const set = new Set();
                 set.add(1);
                 set.add(2);
                 set.add(2); // Qaytarilmaydi
                 console.log(set.has(1)); // true
                 console.log(set.size); // 2
